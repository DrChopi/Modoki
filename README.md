```
Usage: !command [OPTIONS] [ARGS]

One tag the bot respond with random message based on AI.

Commands:
        filter  Add/List/Remove word to filter with replacement string.
        Options:
            add "reg" "replacement"         Add word to filter
            edit "reg" "new_replacment"     Replace or create filter
            remove "reg"                    Remove filter
            list                            List all filter of list

        feed    Add/List/Remove feed to list
        Options:
            add "url"                      Add one feeds to list
            rm "url"                       Remove one or more feed of list
            channel_add "channels"         Add one or more channel to list
            channel_remove "channels"      Remove one or more channel of list
            list                           List all feeds
            list_channels                  List all channels

        citation    Add/List/Remove citation to list
        Options:
            add "citation" "author          Add citation to list
            rm "citation"                   Remove citation of list
            list                            List all citation of list
            random                          Print random citation
        
        proverb    Print random proverb

        nofilter    The following message isn't filtered

        help    Show this panel
```
